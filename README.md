# Warmstart

给 AI agent 做一次"用户画像体检"，生成结构化 prompt，粘贴到任何 AI 工具里，agent 第一次对话就知道你是谁。

---

## 能解决什么问题

你每次开新对话都要重新告诉 AI：
- "我要简短的回复，别废话"
- "直接纠正我，不用先鼓励"
- "帮我写初稿，别只给思路"
- "中文回复，别夹英文"

Warmstart 把这些偏好一次收集完，生成一段结构化 prompt。以后每次新对话粘贴进去，AI 就知道怎么跟你说话。

---

## 适合谁

| 人群 | 场景 |
|------|------|
| 频繁使用多个 AI 工具的人 | 一个画像覆盖 ChatGPT / Claude / Cursor / Hermes / 任何 AI |
| 用 AI 写作的人 | 告诉 AI 你要什么风格、给谁看、怎么帮你 |
| 用 AI 编程的人 | 告诉 AI 你的水平、代码风格、要直接给代码还是给思路 |
| 用 AI 做研究/学习的人 | 告诉 AI 你要什么呈现形式、最大瓶颈是什么 |
| 不确定怎么用 AI 的人 | 选"日常陪伴"，AI 会变成你的聊天搭子 |
| 有特殊需求的人 | 选"自定义"，自由描述你的场景和规则 |

## 使用方式

### 网页版（零安装）

打开 https://diandian1001.github.io/warmstart/ ，3 分钟完成：
1. 选场景
2. 回答 12 道题
3. 复制画像 → 粘贴到任何 AI

### Python 版

```bash
pip install git+https://github.com/diandian1001/warmstart.git
python -m warmstart.prompt
```

## 支持的 AI 工具

Warmstart 生成的是纯文本 prompt，不绑定任何平台。

| 工具 | 用法 |
|------|------|
| ChatGPT | 粘贴到 Custom Instructions |
| Claude | 粘贴到 Project Knowledge 或 System Prompt |
| Cursor / Windsurf | 粘贴到 .cursorrules / .windsurfrules |
| Hermes | 粘贴到 SOUL.md |
| 任何其他 AI | 粘贴到 system prompt 或对话开头 |

---

## 收集了什么信息

| 类别 | 问题数 | 说明 |
|------|--------|------|
| 出生信息 + MBTI | 6 | 选填，用于命盘快照和性格补充 |
| 场景选择 | 1 | 决定后续问什么维度的问题 |
| 通用偏好 | 5 | 语言、反馈风格、回复长度、任务聚焦、AI角色 |
| 场景维度 | 4 | 按场景不同：写作风格/技术水平/生活状态等 |

总共 12 道题，约 3 分钟完成。

## 设计理念

**以量表为底，以命盘为壳**

- 量表（大五人格快速版）提供可量化的用户画像
- MBTI 作为性格补充参考
- 命盘（紫微斗数，开发中）：结合出生信息，未来将在时间维度上给出一定程度的人生阶段预测——比如你当前处于什么大限、这个阶段的核心课题是什么。目前仅收集出生信息，排盘算法尚未实现
- 优先级：场景维度 > 通用偏好 > MBTI > 命盘（未启用）
